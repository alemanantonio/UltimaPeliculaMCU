Próxima Película del MCU

Proyecto PHP que consume una API pública

Descripción del Proyecto

Este proyecto PHP consulta la API pública de "When is the next MCU film" para obtener información sobre la próxima película del Universo Cinemático de Marvel, incluyendo título, fecha de estreno, póster y días restantes.

Código PHP

A continuación se muestra el código PHP que realiza la consulta a la API y procesa los datos: 

// URL de la API que proporciona información sobre la próxima película del MCU
const API_URL = "https://whenisthenextmcufilm.com/api";

// Inicializar una nueva sesión cURL
$ch = curl_init(API_URL);

// Configurar cURL para que devuelva el resultado como string
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

// Ejecutar la petición y guardar el resultado
$res = curl_exec($ch);

// Decodificar la respuesta JSON a un array asociativo
$data = json_decode($res, true);

// Cerrar la sesión cURL
curl_close($ch);
?>

Explicación del Código

¿Por qué usar cURL?

cURL es una librería que permite hacer peticiones HTTP desde PHP. Es la forma más eficiente y flexible de consumir APIs externas, ya que permite configurar cabeceras, métodos HTTP y manejar respuestas complejas.

Proceso de la API

La API devuelve un objeto JSON con toda la información necesaria. Usamos json_decode() para convertirlo en un array asociativo de PHP que podemos manipular fácilmente.

Estructura de los datos

La respuesta de la API contiene estos campos principales:

  • title: Nombre de la próxima película
  • release_date: Fecha de estreno (formato YYYY-MM-DD)
  • days_until: Días restantes para el estreno
  • poster_url: URL del póster oficial

Implementación en el Frontend

Los datos obtenidos del API se muestran en una interfaz HTML con estilos modernos y responsive: 

<!-- Ejemplo de cómo se muestran los datos en HTML -->
<img src="<?= $data["poster_url"]; ?>" alt="<?= $data["title"]; ?> Poster" class="movie-poster">

<h1 class="movie-title"><?= $data["title"]; ?></h1>

<p class="movie-release">Fecha de Estreno: <?= $data["release_date"]; ?></p>

<div class="countdown">Se estrena en <?= $data["days_until"]; ?> días</div>

Características del diseño

La interfaz incluye:

  • Diseño responsive que se adapta a móviles
  • Modo oscuro/claro según preferencias del sistema
  • Efectos hover interactivos
  • Animaciones suaves con transiciones CSS

Posibles Mejoras

1. Cache de la respuesta

Para reducir llamadas a la API, podríamos guardar la respuesta en un archivo JSON local y actualizarlo solo una vez al día.

2. Manejo de errores

Añadir verificación de errores en la petición cURL y un mensaje alternativo si la API falla.

3. Internacionalización

Traducir la interfaz a diferentes idiomas usando arrays asociativos de PHP.
